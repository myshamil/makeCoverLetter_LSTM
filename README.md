## makeCoverLetter_LSTM
* 자기소개서 써주는 프로그램을 만드는 개인 프로젝트입니다.  
* 언어는 Python, 개발환경은 Visual Studio Code이며 LSTM 모델을 사용합니다.  
* 1차적 목표는 완벽하지 않아도 한 항목에 대해 어느정도 말이 되는 문장으로 이루어진 짧은 문단 하나를 출력하는 것입니다.
